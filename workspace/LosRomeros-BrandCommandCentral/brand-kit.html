Section 02

Typographic System

Two typefaces in strict combination. EB Garamond for authority and editorial weight. Inter for data density and UI legibility. No substitutions permitted.

Primary Typeface Headings & Titles
EB Garamond
Weights: Semi-Bold (600), Bold (700)
Tracking: Normal to tight (−0.02em max)
Line Height: 1.1—1.3 for display use
Use For: H1, H2, H3, Pull quotes, Document titles
@import url('https://fonts.googleapis.com/css2?
  family=EB+Garamond:wght@400;600;700&display=swap');
Secondary Typeface Body & UI Elements
Inter
Weights: Regular (400), Medium (500), SemiBold (600)
Tracking: Normal to wide (0.05em max for labels)
Line Height: 1.6—1.7 for body; 1.0 for labels
Use For: Paragraphs, tables, UI controls, metadata
@import url('https://fonts.googleapis.com/css2?
  family=Inter:wght@300;400;500;600;700&display=swap');

Section 03

Logo Guidelines & Restrictions

Dark Background Usage (Preferred)
Approved

Use on all digital dark mode surfaces. Minimum 120px width enforced.

Light Background Usage
Context-Specific

Reserved for print exports, light-mode legal documents, and PDF generation.

Mandatory Rules

  • → Logo must always render at minimum 120px width for digital and 20mm width for print.
  • → Clear space equal to the height of capital "L" in "Los" on all four sides. No imagery or text may enter this zone.
  • → Set typographic elements in EB Garamond Semi-Bold. Must always state "Limited" or "Ltd." explicitly.
  • → Only in Slate 50 on dark backgrounds or Slate 900 on light backgrounds.

Strict Prohibitions

  • ✕ Usage in Crimson, Amber, or any semantic colour. Maintains corporate neutrality.
  • ✕ Rotation, skewing, stretching, or distorting the lockup in any dimension.
  • ✕ Applying drop shadows, outer glows, or gradient overlays to the logo mark.
  • ✕ Usage on complex photographic backgrounds without an isolated clear zone.
Section 04

CSS Implementation

Copy these CSS custom properties directly into any stylesheet to instantly implement the complete Los Romeros design token system.

Root CSS Variables — Copy-Ready
:root {
  /* -- Backgrounds ------------------------ */
  --bg-base:        #020617;   /* slate-950: page backgrounds */
  --bg-card:        #0f172a;   /* slate-900: primary cards */
  --bg-card-alt:    #1e293b;   /* slate-800: secondary cards */

  /* -- Borders ---------------------------- */
  --border:         #334155;   /* slate-700: all rule lines */

  /* -- Text ------------------------------- */
  --text-primary:   #f1f5f9;   /* slate-100: headings, values */
  --text-secondary: #94a3b8;   /* slate-400: body copy */
  --text-muted:     #64748b;   /* slate-500: metadata, dates */

  /* -- Semantic Colours ------------------- */
  --crimson:        #dc2626;   /* HIGH severity alerts */
  --amber:          #d97706;   /* MEDIUM notices */
  --blue:           #2563eb;   /* Links, info badges */
  --green:          #16a34a;   /* Cleared, resolved only */
  --orange:         #ea580c;   /* Section accents */
  --purple:         #a855f7;   /* Gradient terminus */

  /* -- Brand Gradient --------------------- */
    #f59e0b 0%, #ef4444 50%, #a855f7 100%);

  /* -- Typography ------------------------- */
  --font-heading:   'EB Garamond', Georgia, serif;
  --font-body:      'Inter', system-ui, sans-serif;
}
